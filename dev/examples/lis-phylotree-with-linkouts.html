<lis-phylotree-element> with <lis-linkout-element> and <lis-modal-element>

The <lis-phylotree-element> provides a component for drawing interactive phylogenetic trees. In this example, a phytlotree loaded from the LIS GraphQL API is made interactive by making internal nodes collapsible and adding linkouts to leaf node labels.

To demonstrate how to interact with multiple components, the name of the genes in the <lis-phylotree-element>'s leaf nodes are clickable. Clicking on these labels will open an instance of the <lis-modal-element> containing an instance of the <lis-linkout-element>. The <lis-linkout-element> uses the LIS GraphQL API to load links for the gene that was clicked. See the source code for details on how these components interact.
