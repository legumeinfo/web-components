<lis-pagination-element>

The <lis-pagination-element> provides a generic pagination element. When the page changes the component emits a custom 'pageChange' event containing the new page number. The component will automatically disable the previous button when the page attribute/property is 1 and the next button is disabled when the component's hasNext attribute/property is false (false by default) or when the page attribute/property is greater than or equal to the optional numPages attribute/property. The component also supports changing the page programmatically via its previous() and next() methods.

In this example, an event listener is added to the component that prints the page number to the console when the page changes. The component is available in the console via the paginationElement variable. Use this to try changing the page programmatically.
