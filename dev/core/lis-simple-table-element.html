<lis-simple-table-element>

The <lis-simple-table-element> provides a generic component for building tables programmatically. Specifically, the table is built and populated based on objects and lists provided to the component as attributes/properties. The properties are reactive so the table and its contents can be changed dynamically in response to user input. In this example, the table is built and populated using only JavaScript. See the source code for details. The component is available in the console via the tableElement variable. Use this to try dynamically changing the table and its contents.
