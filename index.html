LIS Web Components

Below are links to examples of the components in the LIS Web Components library. User Components are components intended for use by end users and provide rich functionality. Core Components are primitive components from which user components are built. These are generic building blocks and are not intended for use by end users.

User Components:

  • <lis-gene-search-element>
  • <lis-linkout-element>
  • <lis-publication-search-element>
  • <lis-qtl-search-element>
  • <lis-trait-search-element>
  • <lis-phylotree-element>

Core Components:

  • <lis-form-wrapper-element>
  • <lis-modal-element>
  • <lis-pagination-element>
  • <lis-simple-table-element>
